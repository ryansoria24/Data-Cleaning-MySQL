MySQL Data Cleaning Project

Overview

This project focuses on cleaning and preparing raw data using MySQL. The goal is to transform messy data into a clean and reliable dataset ready for analysis.

Tools Used

* MySQL

Data Cleaning Steps

* Removed duplicate records
* Handled missing (NULL) values
* Standardized inconsistent data formats
* Cleaned text fields using SQL functions

Key SQL Techniques

* GROUP BY & HAVING (to identify duplicates)
* TRIM(), REPLACE() (for text cleaning)
* CASE statements (for data transformation)
* UPDATE & DELETE queries




What I Learned

* Importance of clean data in analytics
* Writing efficient SQL queries
* Real-world data cleaning techniques
